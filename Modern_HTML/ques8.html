What is HTML?

HTML stands for HyperText Markup Language. It is the language used to create web pages. HTML uses tags to describe the structure of a web page, such as headings, paragraphs, lists, and images.

What is Semantic HTML?

Semantic HTML uses tags that have a specific meaning, such as header, main, and footer. This makes it easier for browsers and other tools to understand the content of a web page, which can improve accessibility and SEO.

Commonly Used Semantic Tags

  • header: The header of a web page
  • main: The main content of a web page
  • footer: The footer of a web page
  • article: An independent piece of content
  • section: A grouping of content
  • nav: A navigation bar
  • aside: Content that is aside from the main content
